# ViSpa: Visual-Spatial Interactive Fusion for 3D UAV Localization in mmWave Communications

This repository contains a PyTorch implementation of the **ViSpa** framework for 3D UAV localization using **images** and **mmWave signals**. It includes a single training/evaluation script and pretrained weights.  

---

## Contents

- `main.py` – Training and evaluation script for the multimodal fusion network  
- `best_multimodal_modelv3_topk.pth` – Pretrained model weights
